Pulse Detection Circuit for PPG Sensor

April 2025 | ENGR 305: Microelectronics

Overview

Designed and implemented a discrete-component circuit to detect heartbeats from a PPG (Photoplethysmography) sensor signal. The circuit generates a fixed-width output pulse for each heartbeat and drives an LED indicator, simulating pulse feedback.

Technical Details

  • Built and tested the monostable multivibrator using BJTs (Q1, Q2, Q3)
  • Designed the edge detection circuit using a differentiator and diode isolation
  • Conducted LTspice simulations to validate timing behavior (~167 ms Ton)
  • Chose standard resistor and capacitor values based on hand calculations
  • Diagnosed circuit issues with lab technician and adjusted components to minimize oscillations
  • Verified functionality using oscilloscope waveform captures and pulse timing analysis
  • Responds reliably up to 180 BPM and operates from a 5V DC supply

Technologies

LTspice BJT Transistors PPG Sensor Oscilloscope Circuit Design Microelectronics
